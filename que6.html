Lists of Tags.

  1. ul: Defines an unordered list.
  2. p: Defines a paragraph of text.
  3. ul: Defines an unordered list.
  4. table: Defines a table with rows and columns.
  5. br:used to break the line.
  6. li: Defines the list item.
  7. tr: Define the table row.
  8. td: Define the table data.
  9. th: Define the table heading.
